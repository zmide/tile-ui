Tile UI React ships a focused set of foundational UI primitives that are also available through the React registry.

## Available components

- [Button](/docs/components/button)
- [Input](/docs/components/input)
- [Textarea](/docs/components/textarea)
- [Label](/docs/components/label)
- [Card](/docs/components/card)

## Why this set

These primitives cover the majority of layout, action, and form needs required to build higher-level examples and application surfaces.
