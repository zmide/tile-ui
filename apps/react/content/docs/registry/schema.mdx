# Schema

Tile UI React uses the standard shadcn registry schema.

## Item types

- `registry:ui`
- `registry:hook`
- `registry:lib`
- `registry:file`

## Target paths

Files that need stable destination paths, such as SCSS partials, declare a `target` so the consumer project receives a predictable structure.
