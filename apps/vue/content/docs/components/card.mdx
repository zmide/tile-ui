---
title: Card
description: Composable card primitives for headers, content, descriptions, and actions.
---

## Registry install

```bash
pnpm dlx shadcn@latest add @tile-ui/card
```

## Package usage

```ts
import { TCard, TCardHeader, TCardTitle, TCardContent } from '@tile-ui/vue';
```

## Highlights

- Slot-based composition
- Header, content, and footer primitives
- Useful for docs and app surfaces
