---
title: useLocalStorage
description: Persist refs into local or session storage for simple settings and drafts.
---

> Use this composable when a local preference or draft should survive page refreshes without introducing a larger state layer.

## Registry install

```bash
pnpm dlx shadcn@latest add https://vue.tile-ui.dev/r/use-local-storage.json
```

## Usage

```ts
const value = useLocalStorage('theme', 'light');
```

## What it covers

- Local storage reads and writes
- Safe initialization in browser-only contexts
- Ref-based ergonomics that fit standard Vue composition

## Common use cases

- Theme preferences
- Persisted filters
- Draft values in admin tools
