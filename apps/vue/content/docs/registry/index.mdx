The Vue registry mirrors the React registry structure, but each item contains Vue TSX components or composables tailored for Nuxt and Vue applications.

## Registry URLs

```text
https://vue.tileui.zmorg.cn/r/registry.json
https://vue.tileui.zmorg.cn/r/{name}.json
```

## Available items

- `button`
- `card`
- `core`
- `input`
- `label`
- `styles`
- `textarea`
- `use-copy-to-clipboard`
- `use-local-storage`
- `use-media-query`
- `utils`
